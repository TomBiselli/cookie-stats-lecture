## How to use

1. Click the Binder badge below to launch RStudio in your browser
2. No installation or account needed
3. Open the file you want in the bottom-right Files panel
4. Run code with **Ctrl+Enter** (Windows/Linux) or **Cmd+Enter** (Mac)
